# Contributing

Issues, reproducible bug reports, and documentation suggestions are welcome.

Before opening an issue:

1. Confirm the problem occurs on a physical iOS device supported by the demo.
2. Include the iOS and Xcode versions.
3. Remove private scans, images, credentials, and signing information.

Voxelio is not accepting external code contributions yet. This keeps ownership
clear while the project establishes a contributor license agreement suitable
for its dual-licensing model. Pull requests may be used for discussion, but
should not be assumed accepted or merged.
